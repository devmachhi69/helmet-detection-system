# Helmet Detection System

AI-based Helmet Detection System using YOLOv8 and Flask.

## Features
- Detects riders wearing or not wearing helmets
- License plate detection using OCR
- Violation image capture
- Admin dashboard
- PDF report generation

## Tech Stack
Python  
Flask  
YOLOv8 (Ultralytics)  
OpenCV  
EasyOCR  
MySQL / SQLite  

## Project Structure
flask_app/ → Flask web application  
plate_training/ → license plate model training  
requirements.txt → Python dependencies  

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser

http://localhost:5000
